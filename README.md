# Popfly

Make sure your EC2 development instances die after a few hours.

## Heroku config

Popfly needs the following environment variables set in order to function

    AWS_ACCESS_KEY_ID
    AWS_AMI_IMAGE
    AWS_INSTANCE_TYPE
    AWS_KEY_NAME
    AWS_SECRET_ACCESS_KEY
    AWS_SECURITY_GROUP
    AWS_USER
    POPFLY_PASSWORD
